### 功能：
|   | 描述 | 状态 |
|---|-------|-------|
| 1 | hover显示坐标信息 | 完成 |
| 2 | 底部table若入参类型为数组，则根据数组length自动均分单元格 | 完成 |
| 3 | 呼吸行上下错开显示（且每天首次显示在格子上部） | 完成 |
| 4 | 不同折线的点重合时，合并坐标信息 | 完成 |
| 5 | 点击左下角图示，控制对应折线的显示隐藏 | 完成 |
| 6 | 通过update方法，局部更新表格内容 | 未完成 |
